Les particules temporelles

Le créole utilise des particules avant le verbe pour indiquer le temps :

ka = présent habituel/progressif
Ex: "Man ka chèché" = Je cherche

té = passé
Ex: "Man té alé" = J'allais / Je suis allé

kay/ké = futur
Ex: "Man kay alé" = J'irai

Le pluriel

On utilise "sé" avant le nom pour marquer le pluriel :

timoun = enfant
sé timoun = les enfants

Les adjectifs

Les adjectifs se placent APRÈS le nom (sauf ti, vyé, gran) :

an kay bèl = une belle maison
Mais : an ti-moun = un enfant
