Les particules temporelles

Le créole utilise des particules avant le verbe pour indiquer le temps :

ka = présent habituel/progressif

Exemple : "Man ka chèché" = Je cherche

té = passé

Exemple : "Man té alé" = J'allais / Je suis allé

kay/ké = futur

Exemple : "Man kay alé" = J'irai

Les pronoms personnels

Man / Mwen = Je / Moi

Ou / W = Tu / Toi

I / Li = Il / Elle

Nou = Nous

Zot / Yo = Vous / Ils / Elles

Le pluriel

On utilise "sé" avant le nom pour marquer le pluriel :

timoun = enfant

sé timoun = les enfants

fanm = femme

sé fanm = les femmes

Les adjectifs

Les adjectifs se placent APRÈS le nom (sauf ti, vyé, gran) :

an kay bèl = une belle maison

an fanm jenn = une jeune femme

Mais : an ti-moun = un petit enfant

an vyé nonm = un vieil homme

La négation

pa / pon = ne...pas

Exemple : "Man pa sav" = Je ne sais pas

"I pa vini" = Il n'est pas venu

pa...enmen = ne...jamais

Exemple : "Man pa enmen manjé sa" = Je ne mange jamais ça
